# 📊 Task 04: Marketing Campaign ROI & Funnel Insights

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on a marketing campaign dataset to evaluate campaign performance, understand customer acquisition behavior, and identify the most profitable marketing channels.

The analysis emphasizes the **"Why" behind campaign performance** by examining impressions, clicks, leads, conversions, spending, revenue generation, and return on investment (ROI). The goal is to provide data-driven recommendations for future marketing budget allocation.

---

## 🎯 Objective

* Clean and prepare marketing campaign data.
* Analyze campaign performance across different marketing channels.
* Visualize the marketing conversion funnel.
* Calculate and compare channel-wise ROI.
* Generate business insights and strategic recommendations.

---

## 📂 Dataset Description

The dataset contains campaign-level marketing performance metrics:

| Column      | Description                         |
| ----------- | ----------------------------------- |
| Campaign_ID | Unique campaign identifier          |
| Channel     | Marketing channel used              |
| Impressions | Number of times ads were viewed     |
| Clicks      | Number of user clicks               |
| Leads       | Number of leads generated           |
| Conversions | Number of successful conversions    |
| Spend       | Campaign expenditure                |
| Revenue     | Revenue generated from the campaign |

### Marketing Channels Included

* Email Marketing
* Organic Search (SEO)
* Paid Search (SEM)
* Social Media
* Referral Marketing

---

## 🧹 Data Cleaning & Preparation

The following preprocessing steps were performed:

* Identified and handled missing values.
* Verified and corrected data types.
* Checked for inconsistencies in campaign records.
* Prepared the dataset for ROI and funnel analysis.
* Ensured data quality for accurate visualization and reporting.

---

## 🔍 Key Findings

* **Email Marketing** generated the highest revenue and conversion volume.
* **Referral Marketing** achieved the highest ROI with minimal spending.
* **Organic Search (SEO)** delivered strong returns while maintaining cost efficiency.
* **Social Media** generated high visibility but lower conversion efficiency.
* **Paid Search (SEM)** showed moderate performance and requires optimization.

---

## 💡 Business Recommendations

### Increase Investment In:

✅ Email Marketing

✅ Referral Marketing

✅ Organic Search (SEO)

### Optimize:

⚠️ Paid Search (SEM)

⚠️ Social Media Campaigns

### Strategic Focus:

* Prioritize high-performing channels with strong ROI.
* Improve conversion strategies for lower-performing channels.
* Allocate future budgets based on profitability and conversion effectiveness.
